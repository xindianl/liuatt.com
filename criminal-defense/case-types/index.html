Criminal Case Types

刑事案件类型与辩护审查重点

刘信典律师接受各类刑事案件的咨询与委托评估。罪名名称不是分析终点，具体工作仍需回到行为、主观认识、证据、程序和法律适用。

直接回答

本站刑事业务类型不限。常见咨询包括诈骗、盗窃等侵财犯罪，非法吸收公众存款、合同诈骗等经济犯罪，职务犯罪，故意伤害等人身犯罪，涉毒犯罪，危险驾驶、交通肇事以及妨害社会管理秩序犯罪。是否构成犯罪、构成何罪及责任轻重，必须结合个案证据判断。

侵财犯罪经济犯罪职务犯罪人身犯罪涉毒犯罪驾驶与交通社会管理秩序

诈骗、盗窃等侵财犯罪

重点审查财物取得方式、被害人处分行为、非法占有目的形成时间、数额计算、资金流向、退赃退赔及共同犯罪分工。民事违约、经营失败与刑事诈骗不能只看损失结果。

经济犯罪

可能涉及非法吸收公众存款、集资诈骗、合同诈骗、虚开、走私、洗钱及扰乱市场秩序等。应审查业务模式、资金用途、经营真实性、参与层级、违法所得及行政违法与刑事犯罪的边界。

查看经济犯罪案件家属指南 →

职务犯罪

常见争点包括主体身份、职务便利、财物性质、权钱交易、请托事项、共同犯罪、追赃挽损以及留置与刑事诉讼的衔接。口供与客观证据之间是否相互印证尤为重要。

侵犯人身权利犯罪

故意伤害等案件通常需要审查起因、行为先后、工具和打击部位、伤情鉴定、因果关系、正当防卫或过当、共同参与程度以及赔偿谅解情况。

涉毒犯罪

重点关注毒品来源和去向、主观明知、数量认定、称量与鉴定程序、控制下交付、特情介入、电子数据、同案人口供及行为人在交易链条中的地位作用。

危险驾驶与交通肇事

应核对酒精检测程序、血样提取和保管、车辆驾驶事实、道路属性、事故责任、逃逸认定、伤亡因果关系、赔偿谅解及是否存在从重或从宽情节。

妨害社会管理秩序犯罪

罪名跨度较大，可能涉及帮信罪、网络赌博、寻衅滋事、妨害公务、掩饰隐瞒犯罪所得等。需要区分行为对象、明知内容、实际作用、获利、上下游联系及证据合法性。

单位犯罪与共同犯罪

除罪名本身外，还要分析单位意志、利益归属、决策过程，行为人的职责权限、参与时间、分工、控制力、违法所得和退出情况，避免以职位或同案人口供简单推定责任。

共同审查框架

不同罪名都要回答的六类问题

  1. 行为：客观上实施了什么，哪些是本人行为，哪些只是他人陈述；
  2. 主观：是否明知、故意，非法占有目的何时形成；
  3. 证据：口供、书证、电子数据、鉴定意见能否相互印证；
  4. 程序：取证、搜查、扣押、辨认、讯问等程序是否合法；
  5. 责任：主从犯、单位与个人、既遂未遂、数额和后果如何认定；
  6. 量刑：自首、立功、认罪认罚、退赃退赔、谅解等情节是否成立。
